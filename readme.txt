Micro-F - 96k леталка


--------------------------УПРАВЛЕНИЕ--------------------------

WASDQE - управление самолётом по крену/тангажу/рысканию.
Shift, Ctrl - управление тягой двигателя.
F1 - переключение вида от первого/третьего лица.
ESC - выход.
Мышь, стрелки - свободный обзор ( только в режиме от третьего лица ).
Левая лавиша мыши - стрелять из пулемёта.
Правая клавиша мыши - запустить ракету.
Колесо мыши - увеличить/уменьшить угол обзора.


-------------------ПАРАМЕТРЫ КОМАНДНОЙ СТРОКИ-----------------

-h запуск внутриигровой справки по параметром командной строки.
-ws [ширина] [высота] задать размеры окна.
-fs запустить игру в полноэкранном режиме в текущем разрешении экрана.
-no-vs не использовать вертикальную синхронизацию.
-imy инфертировать ось Y мыши при управлении обзором.
-ms-x [чувствительность] задать чувствительность мыши по оси X.
-ms-y [чувствительность] задать чувствительность мыши по оси Y.

-----------------------ИГРОВОЙ ПРОЦЕСС------------------------

Цель игры - добраться до конца уровня ( 0 по компасу ), попутно
набрав как можно большее количество очков и сделать это за как
можно меньшее время. По пути игроку будут мешать вражеские
самолёты, которые нужно сбивать с помощью пулемёта и ракет.
Над рекой раставлены подбираемые предметы, они помогут возместить
запас здоровья, ракет, жизней. Игровое пространство ограничено
силовым полем, при пересечении которого игрок начинает быстро
терять очки здоровья. В случае смерти игрока, он возраждается,
если есть запас жизней, иначе - игра прекращается.


-------------------------МЕНЮ НАСТРОЕК------------------------

В игре есть меню настроек, через которое можно настроить параметры
графики и генерации уровня. Настраиваются время суток, режим
рендеринга, наличие и количество облаков, качество неба, качество
теней. Так-же можно выбрать зерно генерации мира - при одинаковом
зерне получается одинаковый мир.
